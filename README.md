# 材料列表：
## 铝合金型材（我用的欧标，用国标应该也可以，配件也换成国标就行）
欧标3030 900mm 2根  
欧标3030 840mm 2根  
欧标3060 840mm 2根  
欧标3060 240mm 6根  
## 型材配件
直角角码加紧固螺丝 36组（也可以用直角连接条，只要能直角连接就行）  
面板螺丝和T型螺母 若干（螺丝长度根据你用的板材厚度确定，T型螺母亦可换成滑块螺母）  
铝型材3030用 亚克力5mm直角垫片 20个  
30mm 铝型材橡胶封边条 10米  
## 板材
HDPE板 295mm*295mm 9块  
厚度建议10mm以上，非常稳，我用的16mm厚。另外四个箭头的板可以比五个不活动的板稍薄个2-3mm  
如果你不像我体重这么大（100+kg），用厚一点的亚克力或者木板应该也是可以的
最好找厂家先开好孔，我自己拿电钻钻孔钻得手都麻了
## 电子零件
感应条加橡胶套 16个（淘宝找跳舞机感应条套装）  
6mm-10mm宽的窄金属条 若干（贴在感应条橡胶套上，用于触发感应条）  
xh2.54 一分二连接线 12条  
xh2.54 1m延长线 4条  
IO板 1个（用你喜欢的就行，我用的便宜的USB摇杆板）  

# 制作过程
1. 买材料
我主要在淘宝搞定的，铝型材和HDPE板也可以去本地装修市场找  
2. 组装框架
将铝合金型材用直角角码按下图组装为框架  
![框架图纸](/images/Frame.png "Frame")
这是组装好的样子  
![框架图纸](/images/frame-photo.jpg "Frame photo")
3. 在组装好的框架的一面装上橡胶封边条，用于静音和防震  
![框架图纸](/images/frame-back.jpg "Frame")
4. 在框架上安装感应条和金属条，直接将感应条套平的一面粘上纳米胶再贴到框加上即可，然后在感应条套另一面贴上窄金属条
5. 安装四角上的非活动面板
每个非活动面板四个角垫上垫片，再用八个螺丝和T型螺母锁紧
6. 感应条和IO板接线，感应条两个一组用一分二线连接，两组再用一分二线连接，组成一个方向，用延长线接到IO板的按钮上。走线方面可以沿着铝型材的沟来走，最后汇总到左上或右上的非活动面板背面，整理好用胶布贴在框架上或面板背面  
7. IO板接上电脑，在感应条上先放上活动面板，打开windows的USB手柄设置或spicecfg.exe，测试一下踩板触发是否正常
8. 测试正常后，把每个活动面板用八个螺丝和T型螺母固定到框架上，注意不要锁紧，要给面板留出一点活动空间
9. 开玩
