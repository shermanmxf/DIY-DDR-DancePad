# 材料列表：
## 铝合金型材（我用的欧标，用国标应该也可以，配件也换成国标就行）
欧标3030 900mm 2根  
欧标3030 840mm 2根  
欧标3060 840mm 2根  
欧标3060 240mm 6根  
## 型材配件
直角角码加紧固螺丝 36组（也可以用直角连接条，只要能直角连接就行）  
面板螺丝和T型螺母 若干（螺丝长度根据你用的板材厚度确定，T型螺母亦可换成滑块螺母）  
铝型材3030用 亚克力5mm直角垫片 20个  
30mm 铝型材橡胶封边条 10米  
## 板材
HDPE板 295mm*295mm 9块  
厚度建议10mm以上，非常稳，我用的16mm厚。另外四个箭头的板可以比五个不活动的板稍薄个2-3mm  
如果你不像我体重这么大（100+kg），用厚一点的亚克力或者木板应该也是可以的
## 电子零件
感应条加橡胶套 16个（淘宝找跳舞机感应条套装）  
6mm-10mm宽的窄金属条 若干（贴在感应条橡胶套上，用于触发感应条）  
xh2.54 一分二连接线 12条  
xh2.54 1m延长线 4条  
IO板 1个（用你喜欢的就行，我用的便宜的USB摇杆板）
